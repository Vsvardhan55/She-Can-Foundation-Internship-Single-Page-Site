# She Can Foundation – Internship Project

This is a responsive and modern website created for the She Can Foundation Internship Selection Task.

## Project Overview

The website is designed using **HTML5 and CSS3** with a focus on clean UI/UX, responsiveness, and beginner-friendly structure.

It includes:
- Home Section (Hero Banner)
- About Section (Information Cards)
- Join / Call-to-Action Section
- Responsive Navigation Bar
- Modern Footer
